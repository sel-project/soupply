  • Encapsulation

    Fields:

    Name Type When
    info ubyte
    length ushort

    order index triad (info & 0x7F) is greater than or equal to 96
    order channel ubyte (info & 0x7F) is greater than or equal to 96
    split split (info & 0x10) is not equal to 0
    payload bytes
